Short answers.

Why isn't this on the Play Store?

Google's policies around general-purpose Accessibility apps are strict and inconsistently enforced. Rather than spend weeks in review roulette, we ship direct. You get updates first, we skip the 30% tax.

Is it safe to install?

Yes — the APK is signed with our release key (fingerprint on GitHub). You're trusting the same code any Play Store reviewer would see. Source is at github.com/fainir/handy-ai.

What phones does it work on?

Android 11 and up. The Accessibility screenshot API requires API level 30+.

Can it enter passwords or run payments?

Instructions tell it not to, unless you explicitly ask. It's guidance to the model, not a sandbox — treat it accordingly.

Where does my screen content go?

To Anthropic's Claude API, one screenshot per step, only while a task runs. We never store, proxy, or train on it. See the privacy policy.
